### Installation 
- clone this repo  `git clone https://github.com/nedjwab/microverse-Leaderboard.git` on your terminal.
- Type `cd microverse-Leaderboard` to access the project on the terminal.

#### Install dependencies
```
npm install
```
### Build the app
```
npm run build
```
### Run the app

```
npm start
```
